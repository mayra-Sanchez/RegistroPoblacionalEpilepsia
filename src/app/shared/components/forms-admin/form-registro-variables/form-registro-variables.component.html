¿Para qué sirven las opciones?

Al marcar "¿Tiene opciones?" puedes definir una lista de valores que los usuarios podrán seleccionar cuando usen esta variable. Por ejemplo, una variable llamada "Color favorito" puede tener opciones como: Rojo, Azul, Verde.

¿Se puede escoger más de una opción?
Este formulario solo define las opciones disponibles. Si los usuarios podrán seleccionar una o varias opciones, dependerá de cómo se use esta variable en los formularios de captura posteriores.
